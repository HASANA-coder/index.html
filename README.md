# PayTrack - Customer Payment Manager

A modern, light-themed web application for managing customer payments and tracking debts.

## Features

✅ **Dashboard Overview**
- Total active debt across all customers
- Total payments received
- Number of customers in debt
- Highest debtor identification
- Top debtors list

✅ **Customer Management**
- Add new customers with initial payment amounts
- View all customers in a clean table format
- Search customers by name
- Edit customer details (delete or view)

✅ **Payment Tracking**
- Add payments to customer accounts
- Subtract payments (reverse transactions)
- Auto-calculated remaining balances
- Real-time status updates (Pending, Partial, Paid)

✅ **Individual Customer Details**
- View complete payment information
- Quick add/subtract payment interface
- Delete customer option
- Automatic balance calculations

✅ **Light Theme Design**
- Clean, modern interface
- Responsive on all devices
- Smooth animations and transitions
- Intuitive navigation

## Usage

1. **Add a Customer**: Click "+ Add Customer" and enter name and initial amount
2. **Track Payments**: Click the edit icon (📝) on any customer
3. **Add Payment**: Enter amount and click "➕ Add Payment"
4. **Subtract Payment**: Enter amount and click "➖ Subtract"
5. **Search**: Use the search bar to find customers by name
6. **Delete**: Click delete icon or use the delete button in customer details

## Technical Details

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Storage**: LocalStorage (data persists in browser)
- **Responsive**: Mobile, Tablet, Desktop
- **No Dependencies**: Pure vanilla implementation
